ASMR trend

The Age-standardized mortality rate (ASMR) is a statistical measure used in epidemiology to compare disease death rates, particularly when comparing populations with different age structures.

The results show consistent trends in age-standardized mortality from 1990 to 2024 Age-standardized mortality rates differ by sex, with males experiencing higher rates of disease-specific mortality compared to females.

## Introduction

This section examines the  mortality trends in Mexico without disaggregation (by cause or territory),
with the aim of identifying macro patterns and temporal inflection points.


```{r setup-a1, include=FALSE}
 load("H:/Mon Drive/Broni/Projet R Mortality/BDD/Data_clean/National/20260421 asmr/asmr_nac.rda")

library(dplyr)
mort <- asmr_nac |>   mutate(year = as.integer(year))

library(tidyverse); library(scales); library(knitr); library(kableExtra)

palette_sexo <- c("1" = "#2a7a8c", "2" = "#e84855", "Total" = "grey40")

# mort <- expand.grid(

#   cause = c("Enfermedades cardiovasculares", "Tumores malignos",

# ) |> mutate(
#   deaths = rpois(n(), lambda = 500 + 40*(year-2000) + ifelse(sex=="Hombre",120,0)),
#   rate   = deaths / runif(n(), 1e5, 2e6) * 1e5
# )

mort_yr_sex <- mort |> group_by(year, sex) |> summarise(deaths=sum(n_death), .groups="drop")
mort_yr     <- mort |> group_by(year)      |> summarise(deaths=sum(n_death), .groups="drop") |>
               mutate(sex = "Total")
mort_global <- bind_rows(mort_yr_sex, mort_yr)

# ASMR
asmr_male <- asmr_nac|> filter(sex == 1)

asmr_female <- asmr_nac |> filter(sex == 2)
```

## ASMR trend

The Age-standardized mortality rate (ASMR) is a statistical measure used in epidemiology to compare disease death rates, particularly when comparing populations with different age structures.

#### Temporal serie for ASMR

The results show consistent trends in age-standardized mortality from 1990 to 2024
