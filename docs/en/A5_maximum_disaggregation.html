Block A · Level 5 — Health-territory articulation: state × specific cause × sex

Published

May 16, 2026

Block A · Level 5 Variables : year · sex · state · cause_specific

Introduction

This level represents the maximum disaggregation of Block A: it combines all four analytical dimensions (time, sex, territory, and ICD-10 specific cause) to produce high-resolution state–cause epidemiological profiles.

The central question is: which state × cause × sex combinations concentrate the greatest burden of preventable mortality?

Tip

Recommended use
This level is the natural starting point for prioritising public health interventions at the state level. Results are complemented by the spatial analysis in Block B.

Note

Key takeaways from Level ⑤

  • The state × cause × sex combination reveals specific geographic concentrations that remain hidden at more aggregated levels of analysis.
  • Ischemic heart disease and type 2 diabetes dominate in almost every state, but with very different intensities depending on the region.
  • External causes (homicides, accidents) show the highest M/F ratio and are concentrated in northern and north-western states.
  • This level provides the entry point for prioritising public health interventions with maximum territorial specificity.

## Introduction

This level represents the **maximum disaggregation** of Block A: it combines
all four analytical dimensions (time, sex, territory, and ICD-10 specific cause)
to produce high-resolution state–cause epidemiological profiles.

The central question is: **which state × cause × sex combinations concentrate
the greatest burden of preventable mortality?**

::: {.callout-tip}
**Recommended use**  
This level is the natural starting point for prioritising public health
interventions at the state level. Results are complemented by the spatial
analysis in [Block B](B2_spatial_states.qmd).
:::

```{r setup-a5-en, include=FALSE}
library(tidyverse); library(scales); library(ggrepel); library(kableExtra)
options(ggrepel.max.overlaps = Inf)

palette_sex <- c("1" = "#2a7a8c", "2" = "#e84855")

states_10 <- c("Mexico City", "Jalisco", "Nuevo León", "Veracruz",
               "Chiapas", "Oaxaca", "Guanajuato", "Michoacán", "Puebla", "Sonora")

cause_specific_list <- c(
  "Ischemic heart disease", "Heart failure", "Stroke",
  "Breast cancer", "Cervical cancer", "Prostate cancer", "Lung cancer",
  "Type 2 diabetes", "DM complications", "Liver cirrhosis",
  "Pneumonia", "COPD", "COVID-19",
  "Road traffic accidents", "Homicides", "Suicides"
)

set.seed(55)
mort5 <- expand.grid(
  year           = 1998:2024,
  sex            = c("1", "2"),
  state          = states_10,
  cause_specific = cause_specific_list,
  stringsAsFactors = FALSE
) |> mutate(
  deaths = rpois(n(), lambda = 80 + 8*(year-1998) +
                   ifelse(sex == "1", 30, 0) +
                   case_when(
                     cause_specific %in% c("Ischemic heart disease", "Type 2 diabetes") ~ 250,
                     cause_specific %in% c("Homicides", "Road traffic accidents")        ~ 150,
                     cause_specific == "COVID-19" ~ ifelse(year >= 2020, 300, 0),
                     TRUE ~ 40
                   ) +
                   case_when(
                     state == "Mexico City" ~ 200,
                     state == "Chiapas"     ~ -30,
                     TRUE ~ 0
                   )),
  rate = deaths / runif(n(), 5e5, 8e6) * 1e5
)

last_year <- max(mort5$year)
```
