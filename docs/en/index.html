Mortality in Mexico

This site presents a systematic and progressive analysis of mortality in Mexico based on administrative records (INEGI / Ministry of Health), covering the period 1990–2024. The analytical architecture is organised in three complementary blocks:

📈 Block A — Epidemiological analysis
Progressive disaggregation: global trends → causes → specific causes → states → municipalities.

🗺️ Block B — Regional disparities
Multi-scale mapping: state-level distribution, municipal hotspots, regional gradients.

⏳ Block C — Life expectancy
Trends in life expectancy at birth and at selected ages, disaggregated by sex, state, and municipality.

## Preliminary overview

```{r fig-national-overview, fig.cap="Total mortality trends by sex (1990–2024)"}
ggplot(mort_national,
       aes(x = year, y = deaths / 1000, color = sex, group = sex)) +
  geom_line(linewidth = 1.1) +
  geom_point(size = 2) +
  scale_color_manual(values = palette_sex,
                     labels = c("1" = "Men", "2" = "Women")) +
  scale_x_continuous(breaks = seq(1990, 2024, 4)) +
  scale_y_continuous(labels = label_comma(suffix = " k")) +
  labs(
    title    = "Total Mortality in Mexico",
    subtitle = "Number of deaths by sex, 1990–2024",
    x = NULL, y = "Deaths (thousands)", color = "Sex"
  ) +
  theme_minimal(base_size = 13) +
  theme(
    plot.title       = element_text(face = "bold", color = "#1a2744"),
    plot.subtitle    = element_text(color = "#6c757d"),
    legend.position  = "bottom",
    panel.grid.minor = element_blank()
  )
```
