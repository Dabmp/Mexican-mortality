⑤ Máxima desagregación

Bloque A · Nivel 5 — Articulación salud-territorio: estado × causa específica × sexo

Fecha de publicación

19 de mayo de 2026

Bloque A · Nivel 5 Variables : year · sex · state · cause_specific

Introducción

Este nivel constituye la máxima desagregación del Bloque A: combina las cuatro dimensiones analíticas (tiempo, sexo, territorio y causa específica CIE-10) para producir perfiles epidemiológicos estatal-causales de alta resolución.

La pregunta central es: ¿qué combinaciones estado × causa × sexo concentran la mayor carga de mortalidad evitable?

Tip

Uso recomendado
Este nivel es el punto de entrada natural para priorizar intervenciones de salud pública a nivel estatal. Los resultados son complementados por el análisis espacial del Bloque B.

Perfiles estatales por causa específica

  • Heatmap estado × causa específica
  • Ranking estatal por causa
  • Brecha de género por estado y causa
mort5 |>
  filter(year == last_year) |>
  group_by(state, cause_specific) |>
  summarise(rate = mean(rate), .groups = "drop") |>
  mutate(cause_specific = str_wrap(cause_specific, 15),
         state          = str_trunc(state, 16)) |>
  ggplot(aes(x = cause_specific, y = reorder(state, rate), fill = rate)) +
  geom_tile(color = "white", linewidth = 0.4) +
  scale_fill_distiller(palette = "YlOrRd", direction = 1,
                       name = "Tasa\n/100k") +
  labs(
    title    = paste0("Tasa de mortalidad por estado y causa específica (", last_year, ")"),
    subtitle = "Tasa estandarizada por 100 000 habitantes",
    x = NULL, y = NULL
  ) +
  theme_minimal(base_size = 11) +
  theme(axis.text.x    = element_text(angle = 40, hjust = 1, size = 8),
        plot.title      = element_text(face = "bold", color = "#1a2744"),
        plot.subtitle   = element_text(color = "#6c757d"),
        panel.grid      = element_blank())
top_cause <- mort5 |>
  filter(year == last_year) |>
  group_by(cause_specific) |>
  summarise(total = sum(deaths)) |>
  slice_max(total, n = 6) |>
  pull(cause_specific)

mort5 |>
  filter(year == last_year, cause_specific %in% top_cause) |>
  group_by(state, cause_specific) |>
  summarise(rate = mean(rate), .groups = "drop") |>
  mutate(cause_specific = str_wrap(cause_specific, 18)) |>
  ggplot(aes(x = rate, y = reorder(state, rate), fill = rate)) +
  geom_col(alpha = 0.85) +
  facet_wrap(~cause_specific, scales = "free_x", ncol = 3) +
  scale_fill_distiller(palette = "Blues", direction = 1, guide = "none") +
  labs(
    title = "Ranking estatal por las 6 principales causas específicas",
    x = "Tasa / 100 000 hab.", y = NULL
  ) +
  theme_minimal(base_size = 10) +
  theme(plot.title      = element_text(face = "bold", color = "#1a2744"),
        strip.text      = element_text(face = "bold", size = 8),
        panel.grid.minor = element_blank())
mort5 |>
  filter(year == last_year, cause_specific %in% top_cause[1:4]) |>
  group_by(state, sex, cause_specific) |>
  summarise(rate = mean(rate), .groups = "drop") |>
  pivot_wider(names_from = sex, values_from = rate) |>
  mutate(ratio_hm = `1` / `2`,
         cause_specific = str_wrap(cause_specific, 18)) |>
  ggplot(aes(x = ratio_hm, y = reorder(state, ratio_hm),
             color = ratio_hm > 1)) +
  geom_point(size = 3) +
  geom_vline(xintercept = 1, linetype = "dashed", color = "grey50") +
  facet_wrap(~cause_specific, ncol = 2) +
  scale_color_manual(values = c("TRUE" = "#1a2744", "FALSE" = "#e84855"),
                     labels = c("TRUE" = "H > M", "FALSE" = "M > H")) +
  labs(
    title  = "Ratio hombres/mujeres de la tasa de mortalidad por estado y causa",
    x = "Ratio H/M (> 1 = exceso masculino)", y = NULL, color = NULL
  ) +
  theme_minimal(base_size = 10) +
  theme(plot.title      = element_text(face = "bold", color = "#1a2744"),
        strip.text      = element_text(face = "bold"),
        legend.position = "bottom",
        panel.grid.minor = element_blank())

Tendencias temporales por estado y causa

states_sel <- c("Ciudad de México", "Jalisco", "Chiapas", "Nuevo León")

mort5 |>
  filter(state %in% states_sel, cause_specific %in% top_cause[1:3]) |>
  group_by(year, state, sex, cause_specific) |>
  summarise(rate = mean(rate), .groups = "drop") |>
  mutate(cause_specific = str_wrap(cause_specific, 14)) |>
  ggplot(aes(x = year, y = rate, color = sex, group = sex)) +
  geom_line(linewidth = 0.9) +
  facet_grid(state ~ cause_specific, scales = "free_y") +
  scale_color_manual(values = palette_sexo,
                     labels = c("1" = "Hombres", "2" = "Mujeres")) +
  scale_x_continuous(breaks = c(2000, 2010, 2020)) +
  labs(
    title    = "Tendencias temporales: estado × causa específica × sexo",
    subtitle = "Selección de 4 estados y 3 causas principales",
    x = NULL, y = "Tasa / 100 000 hab.", color = "Sexo"
  ) +
  theme_minimal(base_size = 10) +
  theme(plot.title      = element_text(face = "bold", color = "#1a2744"),
        plot.subtitle   = element_text(color = "#6c757d"),
        strip.text      = element_text(face = "bold", size = 8),
        legend.position = "bottom",
        panel.grid.minor = element_blank())

Puntos clave de este nivel

Nota

Lo esencial del nivel ⑤

  • La combinación estado × causa × sexo revela concentraciones geográficas específicas que quedan ocultas en niveles de análisis más agregados.
  • La cardiopatía isquémica y la diabetes tipo 2 dominan en casi todos los estados, pero con intensidades muy distintas según la región.
  • Las causas externas (homicidios, accidentes) presentan el ratio H/M más elevado y una concentración en estados del norte y noroeste.
  • Este nivel es el punto de entrada para priorizar intervenciones de salud pública con máxima especificidad territorial.
