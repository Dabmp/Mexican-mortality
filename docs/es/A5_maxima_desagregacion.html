⑤ Máxima desagregación

Bloque A · Nivel 5 — Articulación salud-territorio: estado × causa específica × sexo

Fecha de publicación

19 de mayo de 2026

Bloque A · Nivel 5 Variables : year · sex · state · cause_specific

Introducción

Este nivel constituye la máxima desagregación del Bloque A: combina las cuatro dimensiones analíticas (tiempo, sexo, territorio y causa específica CIE-10) para producir perfiles epidemiológicos estatal-causales de alta resolución.

La pregunta central es: ¿qué combinaciones estado × causa × sexo concentran la mayor carga de mortalidad evitable?

Tip

Uso recomendado
Este nivel es el punto de entrada natural para priorizar intervenciones de salud pública a nivel estatal. Los resultados son complementados por el análisis espacial del Bloque B.

Puntos clave de este nivel

Nota

Lo esencial del nivel ⑤

  • La combinación estado × causa × sexo revela concentraciones geográficas específicas que quedan ocultas en niveles de análisis más agregados.
  • La cardiopatía isquémica y la diabetes tipo 2 dominan en casi todos los estados, pero con intensidades muy distintas según la región.
  • Las causas externas (homicidios, accidentes) presentan el ratio H/M más elevado y una concentración en estados del norte y noroeste.
  • Este nivel es el punto de entrada para priorizar intervenciones de salud pública con máxima especificidad territorial.

## Introducción

Este nivel constituye la **máxima desagregación** del Bloque A: combina las cuatro
dimensiones analíticas (tiempo, sexo, territorio y causa específica CIE-10) para
producir perfiles epidemiológicos estatal-causales de alta resolución.

La pregunta central es: **¿qué combinaciones estado × causa × sexo concentran
la mayor carga de mortalidad evitable?**

::: {.callout-tip}
**Uso recomendado**  
Este nivel es el punto de entrada natural para priorizar intervenciones de salud pública
a nivel estatal. Los resultados son complementados por el análisis espacial del [Bloque B](B2_espacial_estados.qmd).

```{r setup-a5, include=FALSE}
library(tidyverse); library(scales); library(ggrepel)
options(ggrepel.max.overlaps = Inf)

palette_sexo <- c("1" = "#2a7a8c", "2" = "#e84855")

states_10 <- c("Ciudad de México", "Jalisco", "Nuevo León", "Veracruz",
               "Chiapas", "Oaxaca", "Guanajuato", "Michoacán", "Puebla", "Sonora")

cause_specific_list <- c(
  "Cardiopatía isquémica", "Insuficiencia cardíaca", "EVC",
  "Cáncer de mama", "Cáncer cervical", "Cáncer de próstata", "Cáncer de pulmón",
  "Diabetes tipo 2", "Complicaciones DM", "Cirrosis hepática",
  "Neumonía", "EPOC", "COVID-19",
  "Accidentes de tránsito", "Homicidios", "Suicidios"
)

set.seed(55)
mort5 <- expand.grid(
  year           = 1998:2024,
  sex            = c("1", "2"),
  state          = states_10,
  cause_specific = cause_specific_list,
  stringsAsFactors = FALSE
) |> mutate(
  deaths = rpois(n(), lambda = 80 + 8*(year-1998) +
                   ifelse(sex == "1", 30, 0) +
                   case_when(
                     cause_specific %in% c("Cardiopatía isquémica", "Diabetes tipo 2") ~ 250,
                     cause_specific %in% c("Homicidios", "Accidentes de tránsito")     ~ 150,
                     cause_specific == "COVID-19"                                       ~ ifelse(year >= 2020, 300, 0),
                     TRUE ~ 40
                   ) +
                   case_when(
                     state == "Ciudad de México" ~ 200,
                     state == "Chiapas"          ~ -30,
                     TRUE ~ 0
                   )),
  rate = deaths / runif(n(), 5e5, 8e6) * 1e5
)

last_year <- max(mort5$year)
```

## Puntos clave de este nivel

::: {.callout-note icon="false"}
**Lo esencial del nivel ⑤**

- La combinación estado × causa × sexo revela **concentraciones geográficas específicas**
  que quedan ocultas en niveles de análisis más agregados.
- La **cardiopatía isquémica** y la **diabetes tipo 2** dominan en casi todos los estados,
  pero con intensidades muy distintas según la región.
- Las **causas externas** (homicidios, accidentes) presentan el ratio H/M más elevado
  y una concentración en estados del norte y noroeste.
- Este nivel es el punto de entrada para **priorizar intervenciones** de salud pública
  con máxima especificidad territorial.
:::
