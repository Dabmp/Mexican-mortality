③ bis · Análisis espacial por Municipios

Bloque B — Heterogeneidad intra-estatal y detección de hotspots

Fecha de publicación

21 de mayo de 2026

Bloque B · Nivel 3 bis Variables : municipality · year · cause

Introducción

El nivel municipal constituye la escala más fina del análisis espacial. Permite revelar :

  • L’heterogeneidad intra-estatal : dos municipios del mismo estado pueden tener perfiles de mortalité radicalmente distintos.
  • La existencia de hotspots locales (focos de mortalidad elevada) a menudo invisibles en análisis estatales.
  • El papel de determinantes locales : urbanización, marginación, acceso a servicios.
Advertencia

Nota sobre confidencialidad
Para municipios pequeños (< 5 000 hab.), los datos de mortalidad pueden estar sujetos a restricciones de divulgación (INEGI). Aplicar técnicas de suavizado espacial (kriging, kernel density) en esos casos.

Nota

Méthode recommandée en production

La détection de clusters locaux repose sur la statistique Gi* de Getis-Ord ou sur l’LISA (Local Indicators of Spatial Association) via spdep :

library(spdep)
# Construire la matrice de poids spatiaux
w    <- poly2nb(mx_mun, queen = TRUE)
wl   <- nb2listw(w, style = "W")

# LISA (Moran local)
lm   <- localmoran(mx_mun$rate_total, wl)
mx_mun$local_I <- lm[,1]
mx_mun$p_value <- lm[,5]

# Classifier : HH, LL, HL, LH, ns
mx_mun <- mx_mun |>
  mutate(lisa_class = case_when(
    p_value < 0.05 & local_I > 0 & rate_total > mean(rate_total) ~ "HH (hotspot)",
    p_value < 0.05 & local_I > 0 & rate_total < mean(rate_total) ~ "LL (coldspot)",
    p_value < 0.05 & local_I < 0 ~ "Outlier spatial",
    TRUE ~ "Non significatif"
  ))

```{r setup-b3, include=FALSE}
library(tidyverse); library(scales); library(sf); library(tmap)
tmap_mode("plot")

set.seed(77)
n_mun <- 100
mun_names <- paste0("Municipio_", str_pad(1:n_mun, 3, pad="0"))

# Géométrie simulée (grille) — remplacer par shapefile INEGI
mun_grid <- st_make_grid(
  st_as_sfc(st_bbox(c(xmin=-104,ymin=19,xmax=-98,ymax=22),crs=4326)),
  n=c(10,10), square=TRUE, crs=4326
) |> st_sf() |> mutate(
  municipality = mun_names,
  state        = sample(c("Jalisco","Michoacán","Colima","Guerrero"), n_mun, replace=TRUE),
  pop          = runif(n_mun, 5000, 250000),
  rate_total   = rnorm(n_mun, mean=150, sd=45),
  rate_cv      = rnorm(n_mun, mean=30, sd=12),
  rate_ext     = rnorm(n_mun, mean=25, sd=20),
  margination  = runif(n_mun, 0, 1),
  cluster      = sample(c("Hotspot","Coldspot","No significativo"), n_mun,
                        replace=TRUE, prob=c(0.15,0.15,0.70))
)
```
