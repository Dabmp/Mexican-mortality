Mortalidad en México

Este sitio presenta un análisis sistemático y progresivo de la mortalidad en México a partir de registros administrativos (INEGI / Secretaría de Salud), cubriendo el período 1990–2024. La arquitectura analítica se organiza en tres bloques complementarios:

📈 Bloque A — Análisis epidemiológico
Progresión por desagregación de variables: tendencias globales → causas → causas específicas → estados → municipios.

🗺️ Bloque B — Disparidades regionales
Cartografía multi-escala: distribución por estados, hotspots municipales, gradientes regionales.

⏳ Bloque C — Esperanza de vida
Evolución de la esperanza de vida al nacer y a distintas edades, con desagregación por sexo, entidad y municipio.

Síntesis preliminar

ggplot(mort_nacional,
       aes(x = year, y = deaths / 1000, color = sex, group = sex)) +
  geom_line(linewidth = 1.1) +
  geom_point(size = 2) +
  scale_color_manual(values = palette_sexo,
                     labels = c("1" = "Hombres", "2" = "Mujeres")) +
  scale_x_continuous(breaks = seq(1990, 2024, 4)) +
  scale_y_continuous(labels = label_comma(suffix = " k")) +
  labs(
    title    = "Mortalidad total en México",
    subtitle = "Número de defunciones por sexo, 1990–2024",
    x = NULL, y = "Defunciones (miles)", color = "Sexo"
  ) +
  theme_minimal(base_size = 13) +
  theme(
    plot.title       = element_text(face = "bold", color = "#1a2744"),
    plot.subtitle    = element_text(color = "#6c757d"),
    legend.position  = "bottom",
    panel.grid.minor = element_blank()
  )
